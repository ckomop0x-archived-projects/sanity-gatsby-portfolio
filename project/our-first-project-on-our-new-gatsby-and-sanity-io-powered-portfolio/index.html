Our first project on our new Gatsby and Sanity.io powered portfolio

Welcome to our new portfolio!

What just happened?

We have just added a new repository to our GitHub account. There we have all the code for this portfolio’s frontend made with Gatsby, and Sanity Studio, the environment where we will be editing our content. They are deployed on Netlify. So everything is ready for us to get started with our portfolio! We can even edit content and use the Netlify widget on the Studio’s dashboard to deploy new changes to the web.

Sanity.io lets us work with structured content. That means we can add more content types. Who knows, We might add a blog, maybe a podcast, some products that we need to market, or our bi-weekly newsletter (we should really get on that soon)! We can even connect other frontends, or apps, to our Sanity project. Who knows, one day we can even try one of those voice assistants with it?

But firstly, how could we customize this portfolio to our own branding and such?

We should probably check out the tutorial videos on how to get started with customizing both the content schema and the frontend accordingly. We're sure that we'll find them on the dashboard in the Studio.

There's plenty of cool things to explore:

  • How to customize content schemas
  • How to use structure builder to make own workflows
  • How to manage widgets on dashboard
  • How to turn on real-time content preview in Gatsby

And if we're stuck, we could always seek answers in the friendly developer community.
